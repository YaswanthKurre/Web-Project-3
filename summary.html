Description: The 15 Puzzle is a classic sliding puzzle game where the player rearranges numbered tiles in a grid to achieve a specific order. In this project, a web-based implementation of the 15 Puzzle game is developed using HTML, CSS, and JavaScript. The game involves shuffling tiles and allowing the player to solve the puzzle by sliding the tiles into the correct order.
